[`@lilib/utils`](https://www.npmjs.com/package/@lilib/utils) has some common utils.

## Install

```shell
npm install --save @lilib/utils
```

## Example

```ts
import { warning } from "@lilib/utils";

warning(true, "Some warning message...");
```
